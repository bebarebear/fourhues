FOUR HUES CLICKABLE WEBSITE

How to open:
1. Extract the ZIP file.
2. Open index.html in Chrome, Edge, or any modern browser.

Clickable features included:
- Home, shop dropdown, collections, about, and contact navigation
- Category cards that filter the product section
- Product favorite heart buttons
- Add to Cart buttons
- Clickable cart drawer with total computation and remove buttons
- Subscribe form with email validation
- Footer/social/contact links

Files:
- index.html
- style.css
- script.js
- assets folder with image crops from the provided design reference
